Citric Acid Cycle

Krebs Cycle / TCA Cycle

Acetyl-CoA

Combines with Oxaloacetate to initiate the Citric Acid Cycle.

Citrate

Enzyme: Citrate Synthase

Reaction:
Oxaloacetate + Acetyl-CoA → Citrate

Type:
Condensation Reaction

Isocitrate

Enzyme: Aconitase

Reaction:
Citrate ↔ Isocitrate

Type:
Isomerization

α-Ketoglutarate

Enzyme: Isocitrate Dehydrogenase

Products:
NADH + CO₂

Type:
Oxidative Decarboxylation

Succinyl-CoA

Enzyme: α-Ketoglutarate Dehydrogenase

Products:
NADH + CO₂

Succinate

Enzyme: Succinyl-CoA Synthetase

Products:
GTP Generated

Fumarate

Enzyme: Succinate Dehydrogenase

Products:
FADH₂ Generated

Malate

Enzyme: Fumarase

Reaction:
Fumarate + H₂O → Malate

Oxaloacetate

Enzyme: Malate Dehydrogenase

Products:
NADH Generated
